---

## Project Overview

**Bulky Bookstore** is a web application designed to simplify the process of renting and purchasing books online. The platform provides a user-friendly interface for browsing, renting, or buying books, catering to both individual users and bookstores. It ensures secure transactions and a seamless user experience.

---

## Key Features

- 🔑 **User Authentication**: Secure login and registration system.  
- 📚 **Book Listings**: Browse available books with search and filtering options.  
- 🛒 **Shopping Cart**: Add books to the cart for rental or purchase.  
- 💳 **Secure Payments**: Integrated payment gateways for smooth transactions.  
- 👤 **User Management**: Personal profiles for users and bookstore owners.  
- 🛠 **Admin Dashboard**: Manage users, books, and transactions.

---

## Technologies Used

- **Backend**: ASP.NET Core MVC  
- **Database**: SQL Server with Entity Framework Core  
- **Frontend**: HTML, CSS, JavaScript  
- **Security**: ASP.NET Core Identity
